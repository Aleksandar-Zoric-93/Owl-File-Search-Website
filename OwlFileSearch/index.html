Features that will make your everyday experience easier

Select a Directory

Select a single directory you want to search your file for or select multiple directories using our multi directory select feature

Search

Search all files in the selected directory or search for files based on your search criteria

Open File

You have the option to open the file in the default program or open the file location. When a file is selected, all the file information you will need is displayed

Add to works or favorites

You can add files to your works or favorites list and access it anytime. You will never lose track of your files again

Features

Great features you'll love.

Owl File Search is simple. With features to make your experience easier and if you do not wish to use some features, just disable them and personlise your home screen to display only information you will need. Make it personal

Single or Multi Directory Search

Search for any file using our fast and efficent system. You can search a single directory or decide to search multiple directories at once. Do you want to open the file with your default program or just open the file location? The option is here.

Export files to .CSV .TXT .RTF or .DOC

Large list of files may become difficult to read and manage. Export your list to any of the file formats above and format it to your liking. This can also be useful if you wish to email the list or just keep a record for yourself.

Show or Hide File Information

You may not be interested to see the extra file information. Just disable it on the setting screen and never see it again. Extra file information includes date created, date modified and file size.

Open File in Default Program or File Location

To access the file quickly, you can open it in your default program via Owl File Search and begin making your changes. Do you wish to open folder that the file is contained in? You can do so and we will highlight the selected file for you in the folder.

Search by Partial Name or by File Extension

If you are not sure of the full file name, you can search by the partial file name or by file extension. If the file exists in the selected directory/ies, Owl File Search will find it. No need to change settings to enable different searches, just type the file name or file extension in the search box.

Fast and Efficent Search System

Owl File Search is efficient on how it executes file searches. It does not waste time to get the information you need. We have tested Owl File Search to return 20,000+ files in less than 5 seconds. If you search by giving it a search criteria, it is almost instant.
